# Design Patterns

Repositório com 3 padrões de projeto escolhidos e documentados com base no catálogo do Refactoring Guru.

Padrões incluídos:
- **Criacional:** Singleton
- **Estrutural:** Adapter
- **Comportamental:** Observer

Cada padrão tem:
- explicação do problema que resolve,
- solução e quando aplicar,
- diagrama UML (PlantUML),
- implementação em JavaScript (Node.js),
- exemplo executável e README específico.

## Referência (direitos autorais)
Conteúdo teórico baseados no catálogo do Refactoring.Guru:  
https://refactoring.guru/pt-br/design-patterns

## Uso de LLM
**LLM usada:** ChatGPT - modelo GPT-5.1.

## Como executar os exemplos
1. Tenha Node.js instalado.
2. No diretório do repositório:
```bash

# Singleton
node creational/singleton/index.js

# Adapter
node structural/adapter/index.js

# Observer
node behavioral/observer/index.js
```
